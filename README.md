# Short overview
Time came for another fun guided project from boot.dev and in fact it was indeed really fun. This repo serves as a static site generator from markdown files
Following the project guidelines i did not dare to implement nested markdown support such **as _italic_ inside** bold but I guess maybe one day I'll have the will to come back to it :)

The project is based on custom classes such as HTMLNodes and TextNodes that have necessary methods to turn them into raw html strings to be finally exported as a complete page.
